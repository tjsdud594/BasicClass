
[1] file 관리하는 명령어
1. 학습 순서
    1. file 생성 및 데이터 저장 
    2. file을 이동 및 삭제
    
2. file 생성 및 내용  보기
    방법1 : vi 텍스트 편집기 사용
        $vi test.txt
        i 입력해서 데이터 insert 명령어를 적용
        
        대량의 데이터 작성
        정상 저장 
            esc  -> :wq!
        만일 작업 오류가 생겼을 경우 해결책
            esc 다수 클릭 & ctrl+C & :q!
           
    방법2 : 단순 확인
        $cat text.txt 
    방법3 : 소량의 데이터를 하나의 명령어로 파일 생성
        $echo "데이터" > test.txt        
        
            
    방법4 : 한 line씩 이동이 되면서 확인 가능한 명령어
        $less test.txt    
            방향키(위아래) 또는 enter로 line 단위로 read
            파일 보기 중지
                :q          중지
                :은 이미 존재 따라서 q만 입력하시면 종료
    방법5 : line수 지정해서 확인 가능
        $head -n 5 test2.txt
            첫 머리의 5line 보기 
        $tail -n 2 test2.txt
    방법6 : enter로만 다음 내용 확인 가능 단, 문서의 몇%가 보여지는지 확인
        - 종료시 ctrl+z
        $more test2.txt
    방법7 : 빈파일 생성, 다중 파일 생성도 가능 
        $touch test.txt
        $ touch a.txt b.txt
3. 디렉토리 내용 확인
    1. 단순 확인
        디렉토리? 폴더? 구분이 안 감
        $ls 
    2. 사용자들이 생성한 내용 확인 
        $ls -l
    3. linux 자체의 설정 파일까지도 확인하고 싶은 경우
        $ls -al 
        
        
4. file에 데이터 입력에 따른 명령어 및 키워드
    1. 사용 기본 프로그램이 vi 편집기    
    2. 필수 키워드
        파일 오픈해서 새로운 데이터 추가 작업시에 필수인 표현법 
        1. i - insert의 약자
        2. a - append의 약자
        3. esc -> :wq! 수정된 내용 저장 및 종료
        4. esc -> :q!  수정되었다 하더라도 무시하고 저장없이 종료
        5. DD - 커서가 있는 line의 데이터 삭제 의미 
    3. 발생된 문제 및 해결점
        vi test.txt
            test.txt 파일이 오픈
        입력 시도
            잘된 case / 재선 & 현수씨가 삑삑 입력 불가
            해결책 - a or i 입력
